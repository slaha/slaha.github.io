Usage

To encrypt data with the key, download the key and execute: 

gpg --encrypt --recipient-file "pgp.txt" --output "encrypted" < "plain"
where
  • pgp.txt is downloaded public key,
  • encrypted is name of output file,
  • plain is path to file to encrypt.
